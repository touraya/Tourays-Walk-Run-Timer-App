# Walk & Run Interval Timer

A mobile-first repeating interval timer for walking and running.

## Features

- Set separate walking and running durations
- Automatically alternates until you press Stop
- Voice prompts: “Start walking” and “Start running”
- Alarm beeps and phone vibration
- Pause and resume
- Elapsed time, completed cycles and round counter
- Keeps the screen awake when supported
- Installable as a Progressive Web App
- Works offline after the first visit
- No external libraries or images

## Publish with GitHub Pages

1. Create a new GitHub repository.
2. Upload all files and folders from this project.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`.
6. Save. GitHub will show the public app link.

## Install on iPhone

1. Open the GitHub Pages link in Safari.
2. Tap the Share button.
3. Tap **Add to Home Screen**.
4. Open it from the new home-screen icon.

> On iPhone, keep the app visible for the most reliable spoken prompts and alarms. iOS may restrict browser audio when the screen is locked.
