# Tourays Walk & Run Timer V4 Stable

This version is intentionally self-contained. All styling and app logic are inside `index.html` to prevent GitHub Pages from mixing old HTML, CSS and JavaScript files.

## Upload instructions

1. Upload and replace `index.html`, `manifest.json`, `icon.svg`, and `README.md`.
2. Delete the old `style.css`, `app.js`, and `sw.js` files from the repository.
3. Commit directly to `main`.
4. Wait one or two minutes, then close Safari completely and reopen the app link.

The app automatically unregisters the old service worker and clears its old cache.
