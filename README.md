# Tourays Walk & Run Timer V3

## Main new feature
During an active workout, tap **Change intervals during workout**. You can change walking and running durations without stopping the workout or losing elapsed time, GPS distance, walking/running distance, cycles, calories, or workout history.

You can choose whether the new duration starts immediately for the current interval or only from the next interval.

## Other features
- Manual minutes and seconds
- Quick presets and saved custom workouts
- GPS total, walking and running distance
- Walking and running time
- Current pace and average pace
- Estimated calories
- Circular progress display
- Voice prompts, alarm and vibration
- Pause and resume
- Workout history stored on the device
- Offline support
